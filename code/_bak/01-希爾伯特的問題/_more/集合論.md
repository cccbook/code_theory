# 集合論公理系統

```math
\begin{align*}
& (1)\;  S \in F； \\
& (2)\;  if\; A \in F,\;then\; \bar{A} \in F;  \\
& (3)\;  if\; A \in F, B \in F,\;then\; A \cup B \in F;  \\
\end{align*}
```

## Python 的集合運算示範

```py
from sets import Set
A = Set([1, 3, 5, 7, 9])
B = Set([2, 3, 6, 8, 9])
print "A=", A
print "B=", B
print "A|B=", A|B
print "A&B=", A&B
print "A-B=", A-B
print "B-A=", B-A
```

執行結果

```
A= Set([1, 3, 9, 5, 7])
B= Set([8, 9, 2, 3, 6])
A|B= Set([1, 2, 3, 5, 6, 7, 8, 9])
A&B= Set([9, 3])
A-B= Set([1, 5, 7])
B-A= Set([8, 2, 6])
```
